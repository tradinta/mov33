# **App Name**: Mov33

## Core Features:

- Event Discovery: Browse and filter events, tours, and nightlife experiences based on category, location, and date.
- Seamless Ticketing: Securely purchase tickets through M-Pesa with digital tickets delivered to user profiles and email.
- Personalized Profiles: Manage purchased tickets, loyalty points, and personal information in user profiles.
- Organizer Dashboard: Create and manage events/tours, configure ticket tiers, track attendance, and create promotional codes.
- Influencer Dashboard: Track campaign performance, monitor commissions, and access tracking links for multiple channels.
- Nightclub Dashboard: Manage and promote weekly or special events within the Nightlife section.
- AI Event Recommendation Tool: An AI-powered recommendation system suggests relevant events to users based on their preferences and past activity. This tool leverages machine learning to tailor event recommendations. 

## Style Guidelines:

- Primary color: Deep Blue (#0D1B2A) for the main background, conveying a premium and modern feel.
- Accent color: Burnt Orange (#E76F51) for calls-to-action and highlights to create energy and guide user actions.
- Background color: Light Sand (#F8F5F2) for content panels to create visual contrast.
- Headlines and Logo: 'Montserrat' sans-serif font (Bold / ExtraBold) to establish strong brand presence. Note: currently only Google Fonts are supported.
- Subheads and Buttons: 'Poppins' sans-serif font (Medium / SemiBold) for clear and concise interface elements. Note: currently only Google Fonts are supported.
- Body Text: 'Lato' sans-serif font or 'Open Sans' sans-serif font for readability and accessibility. Note: currently only Google Fonts are supported.
- Simple line-based icons with a consistent stroke weight for a minimalistic and modern look.
- Hero banners with overlay gradients ensure text readability while maintaining visually engaging imagery.
- Clean card-based content sections with hover interactivity to promote engagement.
- Subtle hover animations and micro-interactions provide a smooth and modern user experience.
